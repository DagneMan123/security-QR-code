SECURITY MANAGEMENT SYSTEM - POSTGRESQL SETUP

System now uses PostgreSQL with Sequelize ORM

================================================================================
STEP 1: INSTALL POSTGRESQL
================================================================================

Windows:
  1. Download: https://www.postgresql.org/download/windows/
  2. Run installer
  3. Remember password for 'postgres' user
  4. Default port: 5432

macOS:
  brew install postgresql@15
  brew services start postgresql@15

Linux:
  sudo apt-get install postgresql postgresql-contrib
  sudo systemctl start postgresql

================================================================================
STEP 2: CREATE DATABASE
================================================================================

Open Command Prompt/Terminal and run:

psql -U postgres

Then in psql:

CREATE DATABASE security_db;

\q

================================================================================
STEP 3: UPDATE .env FILE
================================================================================

Edit: backend/.env

Set these values:

PORT=5000
NODE_ENV=development

DB_HOST=localhost
DB_PORT=5432
DB_NAME=security_db
DB_USER=postgres
DB_PASSWORD=postgres

JWT_SECRET=smart_security_jwt_secret_key_12345
ADMIN_EMAIL=security@gmail.com
ADMIN_PASSWORD=security
QR_SECRET=smart_security_key_123

IMPORTANT: Change DB_PASSWORD to your PostgreSQL password!

================================================================================
STEP 4: INSTALL BACKEND DEPENDENCIES
================================================================================

cd backend
npm install

================================================================================
STEP 5: START BACKEND
================================================================================

npm run dev

You should see:
[nodemon] starting `node server.js`
Server running on port 5000

✅ Backend is running!

================================================================================
TROUBLESHOOTING
================================================================================

PostgreSQL Not Running?

Windows:
  pg_ctl -D "C:\Program Files\PostgreSQL\15\data" start

macOS:
  brew services start postgresql@15

Linux:
  sudo systemctl start postgresql

Connection Error?

Check .env file has correct values:
  DB_HOST=localhost
  DB_PORT=5432
  DB_NAME=security_db
  DB_USER=postgres
  DB_PASSWORD=your_password

Database Not Found?

Create it:
  psql -U postgres
  CREATE DATABASE security_db;
  \q

================================================================================
VERIFY CONNECTION
================================================================================

Test connection:

psql -U postgres -d security_db

You should see:
psql (15.0)
Type "help" for help.
security_db=#

Exit:
\q

QUICK COMMANDS
================================================================================

# Start PostgreSQL
psql -U postgres

# Create database
CREATE DATABASE security_db;

# List databases
\l

# Connect to database
\c security_db

# List tables
\dt

# Exit
\q

# Backup database
pg_dump -U postgres security_db > backup.sql

# Restore database
psql -U postgres security_db < backup.sql
