================================================================================
COMMON ERRORS & FIXES
================================================================================

Error: "Cannot find package 'pg'"
Fix: Delete node_modules and package-lock.json, then npm install

Error: "Port 5000 already in use"
Fix: Kill process on port 5000 (see troubleshooting commands above)

Error: "npm install fails"
Fix: npm cache clean --force, then npm install

Error: "Database connection error"
Fix: Check .env has DB_DIALECT="sqlite"

Error: "Module not found"
Fix: Make sure you're in correct directory (cd backend, cd admin, etc)
