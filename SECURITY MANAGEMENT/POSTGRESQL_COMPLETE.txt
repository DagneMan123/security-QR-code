QUICK START (15 MINUTES)
================================================================================

1. INSTALL POSTGRESQL
   Windows: https://www.postgresql.org/download/windows/
   macOS: brew install postgresql@15
   Linux: sudo apt-get install postgresql

2. CREATE DATABASE
   psql -U postgres
   CREATE DATABASE security_db;
   \q

3. UPDATE .env
   DB_HOST=localhost
   DB_PORT=5432
   DB_NAME=security_db
   DB_USER=postgres
   DB_PASSWORD=postgres

4. INSTALL BACKEND
   cd backend
   npm install

5. START BACKEND
   npm run dev

6. START ADMIN (New Terminal)
   cd admin
   npm install
   npm run dev

7. START SECURITY (New Terminal)
   cd security-fronend
   npm install
   npm run dev

8. ACCESS
   Admin: http://localhost:5173
   Security: http://localhost:5174

================================================================================
TROUBLESHOOTING
================================================================================

PostgreSQL Not Running?
  Windows: pg_ctl -D "C:\Program Files\PostgreSQL\15\data" start
  macOS: brew services start postgresql@15
  Linux: sudo systemctl start postgresql

Connection Error?
  Check .env file has correct values
  Make sure PostgreSQL is running
  Verify password is correct

Database Not Found?
  psql -U postgres
  CREATE DATABASE security_db;
  \q

npm install Fails?
  npm cache clean --force
  npm install

================================================================================
USEFUL COMMANDS
================================================================================

# Connect to database
psql -U postgres -d security_db

# List databases
psql -U postgres -l

# List tables
psql -U postgres -d security_db -c "\dt"

# Backup database
pg_dump -U postgres security_db > backup.sql

# Restore database
psql -U postgres security_db < backup.sql

# Create user
psql -U postgres
CREATE USER security_user WITH PASSWORD 'password';
GRANT ALL PRIVILEGES ON DATABASE security_db TO security_user;
\q
